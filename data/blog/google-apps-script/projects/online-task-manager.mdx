## Overview


 <PayButton props="Buy" href="https://buy.stripe.com/3cs8wM4brajn0r6cMM"/>
This Google Spreadsheet based task manager allows accountant to manage their Client's Tasks, create it's related Timesheet entries.

User can add different Task Types, Clients & Activity Types.

Then the user can create tasks and schedule it's reminder and deadline date.
The created tasks then can be tracked in Upcoming Tasks view.
In Upcoming Tasks view the user can mark the task to completed.
The task can also me marked completed and recreated automatically with the set periodicity.
